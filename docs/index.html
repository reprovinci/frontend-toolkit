Typography

The base typography is based on a scaling font size with a min and max size. The default font-size on the <html> is clamp(14px, 2vw, 18px). Try scaling up and down this browser viewport and see the result.

Text styles

There 2 text stylings. Copy text, or body text, is in a .text class which makes it possible to style the copy type apart from the UI type.
